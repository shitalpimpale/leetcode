## Usage

### Running Individual Solutions

```javascript
const solutions = require('./easy-problems-solutions.js');

// Example: Two Sum
console.log(solutions.twoSum([2, 7, 11, 15], 9)); // Output: [0, 1]

// Example: Valid Parentheses
console.log(solutions.isValid('()')); // Output: true

// Example: Climbing Stairs
console.log(solutions.climbStairs(5)); // Output: 8
```

### Testing Examples

```javascript
const {
    twoSum,
    isPalindrome,
    romanToInt,
    climbStairs,
    maxProfit,
    fizzBuzz
} = require('./easy-problems-solutions.js');

// Two Sum
console.log(twoSum([2, 7, 11, 15], 9)); // [0, 1]

// Palindrome Number
console.log(isPalindrome(121)); // true
console.log(isPalindrome(-121)); // false

// Roman to Integer
console.log(romanToInt('MCMXCIV')); // 1994

// Climbing Stairs
console.log(climbStairs(3)); // 3

// Best Time to Buy and Sell Stock
console.log(maxProfit([7, 1, 5, 3, 6, 4])); // 5

// Fizz Buzz
console.log(fizzBuzz(15));
```

- **Tree** - Max Depth, Same Tree, Invert Tree, etc.
- **Math** - Sqrt, Climbing Stairs, Factorial Trailing Zeroes, etc.

- **Dynamic Programming** - Climbing Stairs, Best Time to Buy Sell Stock, etc.

- **Binary Search** - Search Insert Position, First Bad Version, etc.

- **Greedy** - Assign Cookies, etc.

Most solutions are optimized for:
- **Time Complexity**: O(n) or O(log n) where possible
- **Space Complexity**: O(1) or O(n) depending on the problem requirements
